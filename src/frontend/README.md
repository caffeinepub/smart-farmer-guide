# Smart Farmer Guide - Progressive Web App

A mobile-friendly web application designed for Indian farmers to help with daily farming decisions including crop schedules, fertilizer calculations, pest management, weather advisory, market prices, and government schemes.

## Features

- **Crop Schedule**: Step-by-step timeline for Cotton, Soybean, Wheat, and Rice
- **Fertilizer Calculator**: Calculate fertilizer requirements based on crop and land area
- **Pest & Disease Management**: Comprehensive guide with symptoms, control methods, and chemical recommendations
- **Weather Advisory**: Farming recommendations based on weather conditions
- **Market Prices**: Current APMC-style crop prices
- **Government Schemes**: Information about major agricultural schemes

## Technology Stack

- **Frontend**: React + TypeScript
- **Backend**: Motoko (Internet Computer)
- **Styling**: Tailwind CSS with custom agricultural theme
- **UI Components**: shadcn/ui
- **Routing**: TanStack Router
- **PWA**: Service Worker for offline functionality

## Design Philosophy

- **Mobile-First**: Optimized for low-end Android devices
- **Large UI Elements**: Touch-friendly buttons and fonts (minimum 48x48px touch targets)
- **Agricultural Theme**: Green and earthy color palette
- **Simple Navigation**: Icon-based navigation suitable for first-time smartphone users
- **Offline-First**: All static data works without internet connection
